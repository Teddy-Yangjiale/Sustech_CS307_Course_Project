# 项目结构说明

本项目是 Java 17 + Maven 的简易数据库系统项目，入口类是 `edu.sustech.cs307.DBEntry`。

## 根目录

```text
project2/
  pom.xml
  README.md
  src/
  tutorials/
  我的工作说明.md
  项目结构说明.md
```

- `pom.xml`：Maven 配置，包含 JSQLParser、Netty ByteBuf、tinylog、JUnit 等依赖。
- `README.md`：项目基础说明和示例 SQL。
- `tutorials/`：课程讲义。
- `src/main/java/`：数据库系统源码。
- `src/test/java/`：JUnit 测试。

## 源码模块

```text

```

## 模块说明

- `DBEntry.java`：命令行入口，初始化 `DBManager` 并执行 SQL。
- `exception/`：统一异常定义。
- `storage/`：磁盘页、BufferPool、LRU/Clock 页面替换。
- `record/`：定长记录文件、RID、bitmap、record page。
- `meta/`：表结构、列信息、元数据持久化。
- `logicalOperator/`：逻辑算子，包括 scan/filter/project/join/order/aggregate/delete/subquery filter。
- `logicalOperator/ddl/`：DDL 和命令执行器。
- `optimizer/`：`LogicalPlanner` 和 `PhysicalPlanner`。
- `physicalOperator/`：物理算子，包括 seq scan、filter、project、insert、update、delete、count、join、order、aggregate、subquery filter。
- `system/`：`DBManager`、`RecordManager`、`TransactionManager`。
- `tuple/`：`TableTuple`、`ProjectTuple`、`JoinTuple`、`TempTuple`。
- `value/`：`Value`、`ValueType`、`ValueComparer`。
- `index/`：索引接口和内存有序索引骨架，目前 Task 3 未完成。

## 测试模块

```text
src/test/java/
  meta/
  record/
  storage/
  system/
  value/
```

- `storage/`：DiskManager、BufferPool、LRU、Clock 测试。
- `record/`：记录文件和 bitmap 测试。
- `meta/`：元数据测试。
- `system/`：事务测试、Task2 基础功能测试、Task2 高级功能测试。
- `value/`：值比较测试。

## 运行方式

建议用 IntelliJ IDEA 打开 `project2` 目录，等待 Maven 依赖导入后运行 `DBEntry`。

命令行可以使用 IDEA 自带 Maven：

```powershell
& 'D:\download\IntelliJ IDEA Community Edition 2024.3.3\plugins\maven\lib\maven3\bin\mvn.cmd' test
```
