# 项目结构说明

本项目是 Java 17 + Maven 的简易数据库系统项目，入口类是 `edu.sustech.cs307.DBEntry`。

## 根目录

```text
engine-project/
  pom.xml
  README.md
  src/
  tutorials/
```

- `pom.xml`：Maven 配置，包含 JSQLParser、Netty ByteBuf、tinylog、JUnit 等依赖。
- `README.md`：项目基础说明和示例 SQL。
- `tutorials/`：课程讲义，包括 LRU、存储设计、逻辑算子、物理算子、B+ 树索引等。
- `src/main/java/`：数据库系统源码。
- `src/test/java/`：JUnit 测试。

## 源码模块

```text
src/main/java/edu/sustech/cs307/
  DBEntry.java
  exception/
  index/
  logicalOperator/
  meta/
  optimizer/
  physicalOperator/
  record/
  storage/
  system/
  tuple/
  value/
```

### `DBEntry.java`

命令行入口。负责初始化 `DBManager`，读取用户输入 SQL，调用逻辑计划、物理计划并输出结果。

### `exception/`

数据库异常定义。

- `DBException`：统一异常类。
- `ExceptionTypes`：错误类型和错误信息构造。

### `storage/`

页式存储和缓冲池。

- `DiskManager`：负责磁盘文件读写、页面分配、磁盘元数据保存。
- `BufferPool`：负责缓存页面、pin/unpin、flush、页面替换。
- `Page`、`PagePosition`：页对象和页位置。
- `storage/replacer/`：页面替换策略，包括 LRU 和 Clock。

### `record/`

记录文件层。

- 管理定长记录、记录页 header、bitmap、RID。
- 上层表数据最终落到这里。

### `meta/`

元数据管理。

- `MetaManager`：维护表名到表元数据的映射，并保存到 JSON。
- `TableMeta`：表结构、列、索引信息。
- `ColumnMeta`：列名、类型、长度、偏移量。
- `TabCol`：表列组合标识。

### `logicalOperator/`

逻辑算子层。

- 表扫描、过滤、投影、插入、更新、Join 等逻辑节点。
- `logicalOperator/ddl/` 中放 DDL 或命令执行器，例如 `CREATE TABLE`、`SHOW`、`EXPLAIN`。

### `optimizer/`

SQL 到执行计划的转换。

- `LogicalPlanner`：使用 JSQLParser 把 SQL 转成逻辑算子或直接执行事务/DDL 命令。
- `PhysicalPlanner`：把逻辑算子转成物理算子。

### `physicalOperator/`

物理执行算子。

- `SeqScanOperator`：顺序扫描。
- `FilterOperator`：条件过滤。
- `ProjectOperator`：列投影。
- `InsertOperator`、`UpdateOperator`：数据修改。
- `NestedLoopJoinOperator`：嵌套循环 Join 骨架。
- `IndexScanOperator`、`InMemoryIndexScanOperator`：索引扫描骨架。

### `system/`

系统级管理。

- `DBManager`：数据库总控对象，持有 storage、record、meta、transaction 等核心组件。
- `RecordManager`：记录文件管理入口。
- `TransactionManager`：事务 begin/commit/rollback/savepoint 管理。

### `tuple/`

执行结果元组。

- `Tuple`：抽象元组。
- `TableTuple`：来自表记录的元组。
- `ProjectTuple`：投影后的元组。
- `JoinTuple`：Join 结果元组。
- `TempTuple`：临时元组。

### `value/`

值和类型系统。

- `Value`：数据库值对象。
- `ValueType`：支持的类型。
- `ValueComparer`：值比较逻辑。

### `index/`

索引接口和内存有序索引骨架。

- `Index`
- `InMemoryOrderedIndex`

## 测试模块

```text
src/test/java/
  meta/
  record/
  storage/
  system/
  value/
```

- `storage/`：DiskManager、BufferPool、LRU、Clock 等测试。
- `record/`：记录文件和 bitmap 测试。
- `meta/`：元数据测试。
- `system/`：事务测试。
- `value/`：值比较测试。

## 运行方式

建议用 IntelliJ IDEA 打开项目根目录 `engine-project`，等待 Maven 依赖导入后运行 `DBEntry`。

如果命令行有 Maven，可以运行：

```powershell
mvn test
```

当前机器没有全局 `mvn`，可使用 IDEA 自带 Maven：

```powershell
& 'D:\download\IntelliJ IDEA Community Edition 2024.3.3\plugins\maven\lib\maven3\bin\mvn.cmd' test
```
